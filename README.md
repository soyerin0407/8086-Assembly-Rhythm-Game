# ASCII Rhythm Game (8086 Assembly)

A simple rhythm game implemented in 8086 Assembly Language using EMU8086.

## Features

- Three-lane rhythm system
- Random note generation
- Score system
- Combo system
- ESC key to exit
- Start menu

## Controls

| Key | Action |
|------|--------|
| ← | Left lane |
| Space | Center lane |
| → | Right lane |
| ESC | Exit |

## Development Environment

- 8086 Assembly Language
- EMU8086
- DOS Interrupts
- BIOS Interrupts
